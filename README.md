# 🚀 AI Digital Product Manager

A production-grade **Multi-Agent AI System** that converts a startup idea into a complete product plan.

---

## 🎯 Features

- 🧠 Multi-Agent System (LangGraph)
- ⚡ Async FastAPI backend
- 📊 Market Research Generator
- 🧩 Feature Planning (MVP + Full)
- 🗺️ Product Roadmap Generator
- 📄 PRD Generator
- 🏗️ Technical Architecture Suggestions
- 🛠 MCP-style Tool Integration
- 🗄 PostgreSQL (async SQLAlchemy 2.0)
- 🔁 Stateful Graph Workflows
